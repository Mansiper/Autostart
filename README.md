# Autostart

Тут это просто для истории.

Идея: создание программы автозапуска для CD/DVD.

Одна программа создаёт файл с описанием содержимого диска и ссылками на нужные файлы.  
Вторая программа с созданным файлом описания сохраняется на диск с остальными файлами.  
При запуске диска программа автоматически запускается и даёт быстрый доступ к прочему содержимому.

В репозитории лежат два .exe с последней рабочей версией от 19 августа 2009 года.
